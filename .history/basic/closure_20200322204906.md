# 闭包

闭包指的是函数与他周边环境捆绑，在这个环境中，函数可以访问到函数以外的闭包作用域中的变量，在js中，每个函数都是一个闭包

## 作用域

先来看下下面这个例子

```javascript

function test() {
    var a = 123;
    return function() {
        console.log(a);
    }
}

const b = test();
b(); // 123

```

这是一个最简单的闭包的例子，在`test函数`中返回出一个匿名函数，然后调用可以取到`test函数`中的变量
