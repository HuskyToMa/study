至今为止，2020年初，数组的方法已经非常的多了

在原型链上的方法有非常之多，一共可以分为以下几类

## Array 静态方法

### from 

from能够将一个具有类似数组的数据结构转换成数组

> 类数组结构

1. 存在length
2. 存在0,1,2,3,4的key

典型的内容就是`arguments`, function的参数都是可以当做是一个类数组`Array.from(arguments)`;

同时还可能会出现以下问题

```javascript

const obj = {
    1: '1',
    2: '2',
    length: 2
}

console.log(Array.from(obj)); // ['', '1']

```
