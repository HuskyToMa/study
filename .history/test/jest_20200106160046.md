## jest

jest的测试方式

### 安装

`npm insall jest -D`

或

`npm install jest -g`

建议本地还是安装一个jest，因为可以通过 `jest --init` 来初始化jest的config文件

### 使用方式

1. test

``` javascript

test('test a + b ', () => {
    expect(2 + 2).toBe(4);
})

```

test内部有各种方法可以供自己去做对比

- toBe
- toEqual
- toBeNull
- toBeUndefined
- toBeDefined
- toBeTruthy
- toBeFalsy
