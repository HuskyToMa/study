## webpack plugin

根据钩子（hook）和tap方法，插件可以以多种不同的方式运行,针对不同的钩子有不同的调用方法

1. 同步钩子，使用tap

``` javascript

    compiler.hooks.compile.tap('myPlugin', params => {
        console.log('同步方式触发');
    })

```

2. 异步钩子，使用tapAsync或者tapPromise

``` javascript

    compilter.hooks.run.tapAsync('myPlugin', (compilter, callback) => {
        console.log('异步钩子');
        callback();
    })

```
